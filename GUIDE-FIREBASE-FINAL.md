# 🔥 Guide Firebase - Synchronisation Cloud Automatique

## 🎉 Félicitations ! Votre Firebase est configuré

Votre dashboard utilise maintenant **votre propre base de données Firebase** pour la synchronisation cloud.

## ⚡ Synchronisation Instantanée

### 1. Créer votre Code Personnel
- Cliquez "☁️ Sync" dans le dashboard
- Créez un code unique (ex: "MonTradingPro2024")
- **Important** : Même code = mêmes données sur tous appareils

### 2. Première Sauvegarde
- Entrez votre code personnel
- Cliquez "⬆️ Sauvegarder Cloud"
- Vos données sont maintenant sur Firebase !

### 3. Synchroniser sur Nouvel Appareil
- Ouvrez le dashboard sur mobile/autre PC
- Cliquez "☁️ Sync"
- Entrez le MÊME code
- Cliquez "⬇️ Télécharger Cloud"
- **Instantané !** Toutes vos données apparaissent

## 🆘 Résolution de Problèmes

### "Firebase non disponible"
- Vérifiez votre connexion internet
- Utilisez la sauvegarde fichier en attendant

### "Erreur lors de la sauvegarde"
- Code trop simple → Utilisez un code plus complexe
- Connexion instable → Réessayez

### "Aucune donnée trouvée"
- Vérifiez l'orthographe de votre code
- Le code est sensible à la casse
